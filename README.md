# prediksi-harga-laptop-berdasarkan-spesifikasi

Project Machine Learning untuk memprediksi harga laptop berdasarkan spesifikasi.

## Fitur
- Processor Speed
- RAM Size
- Storage Capacity
- Screen Size
- Weight
- Brand
## Target
- Price

## Model
- Linear Regression
- Decision Tree
- Random Forest

## Deployment
Menggunakan Streamlit
